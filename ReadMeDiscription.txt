Online Printing Order Management

This prject provides us a way to give print order through online and allow us to pay online.
We can upload the printing file either in image formats or pdf format.
We can draw our project file with the tools given there in the workspace(Future Improvement).


NOTE : You can see the design for home and gallery web pages in sample folder. Just download the
file in sample images to get a quick overview.



Languages used - HTML\CSS\JavaScript
Software Used  - NetBeans\Atom-TextEditor
Database       - MsAccess
